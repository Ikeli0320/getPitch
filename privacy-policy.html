getPitch — Privacy Policy / 隱私權政策

Last updated: 2026-04-04

English

Data Collection

getPitch does not collect, transmit, or share any personal data. All audio analysis is performed locally in your browser using the Web Audio API.

What the Extension Accesses

  • Audio from the active YouTube tab — analysed in real-time in the browser. No audio data is stored or transmitted anywhere.
  • Tab URL and title — used solely to verify you are on a YouTube watch page and to display the song title in the popup. This data is never stored or sent off-device.
  • chrome.storage.session — stores temporary analysis results (detected key, highest note, BPM) for the current browser session only. This data is erased automatically when the browser closes.
  • chrome.storage.local — stores a single user preference: the transpose slider position (a number from −3 to +3). This value never leaves your device and contains no personal information.

Third-Party Services

getPitch does not use analytics, advertising, or any third-party services. It makes no network requests.

Permissions Explained

  • activeTab — to access the audio element on the current YouTube tab.
  • storage — to pass analysis results between the content script and popup using session storage.
  • tabs — to read the tab URL and title.
  • *://*.youtube.com/* — content script is limited to YouTube only.

Contact

If you have questions about this privacy policy, please open an issue on the project's GitHub repository.

中文

資料蒐集

getPitch 不蒐集、不傳輸、不分享任何個人資料。所有音訊分析均在您的瀏覽器本機進行，使用 Web Audio API，不涉及任何外部伺服器。

本擴充功能存取的內容

  • 目前 YouTube 分頁的音訊 — 於瀏覽器本機即時分析，音訊資料不會被儲存或傳送。
  • 分頁的網址與標題 — 僅用於確認您正在 YouTube 觀看頁面，以及在彈出視窗顯示歌曲名稱。此資訊不會被儲存或上傳。
  • chrome.storage.session — 儲存目前瀏覽器工作階段的暫時性分析結果（調號、最高音、拍速）。瀏覽器關閉後自動清除。
  • chrome.storage.local — 僅儲存一項使用者偏好設定：微調滑條的位置（−3 到 +3 之間的數字）。此資料不會離開您的裝置，不含任何個人資料。

第三方服務

getPitch 不使用任何分析、廣告或第三方服務，也不進行任何網路請求。

權限說明

  • activeTab — 存取目前 YouTube 分頁的音訊元素。
  • storage — 透過 session storage 在內容腳本與彈出視窗之間傳遞分析結果。
  • tabs — 讀取分頁網址與標題。
  • *://*.youtube.com/* — 內容腳本僅限 YouTube 網站執行。

聯絡方式

如有任何關於隱私權政策的疑問，請在本專案的 GitHub repository 開啟 issue。
